# DateBase

La base de datos se generó a partir de JSON.
Se generaron dos archivos en `src/main/resources/db/`:
    -usersDB.json: Se utiliza para el respositorio de Users contiene usuarios del 1001 al 1010.
    -postsDb.json: Se utiliza para el repositorio de Posts, solo contiene posts para los usuarios 1001,1002,1003.
